# ASCII Art Web

## Description

ASCII Art Web is a Go web application that provides a graphical interface for generating ASCII art from text.

The user can:

- enter text in a form
- choose one of the available banner styles
- send a POST request to the server
- view the generated ASCII art in the browser

Supported banner styles:

- standard
- shadow
- thinkertoy

The project is built with Go, HTML templates, and a simple HTTP server structure using handlers, router, middleware, and rendering utilities.
