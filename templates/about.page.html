About The Project

💡

Project Overview

This project is a web version of ASCII Art. It lets users enter text in the browser and convert it into styled ASCII output using a simple.

⚙️

How It Works

The user submits text and selects a banner. The Go server processes the request, generates the ASCII art output, and returns the result

🎨

Available Banners

The project supports three banner styles: standard, shadow, and thinkertoy. Each one gives the same text a different ASCII art.

🌐

HTTP Endpoints

The app uses GET / for the main page and POST /ascii-art for form submission. These routes handle rendering and generating the final ASCII.

🛡️

Error Handling

The server returns proper status codes for valid and invalid requests. It handles missing files, bad input, and unexpected internal errors.

💻

Tech Stack

The project is built with Go, HTML templates, and CSS only. It uses standard Go packages without any external frameworks.
